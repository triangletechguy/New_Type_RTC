# Assets Needed For Final UI Polish

Use this checklist for the high-quality BuzzCast-style visual pass. Put final files in a shared folder or send links with usage rights.

## Brand

- Primary logo: SVG preferred, plus PNG fallback.
- App icon/favicon: SVG or 1024x1024 PNG.
- Brand colors: primary, secondary, accent, success, warning, danger, dark background.
- Typography preference: font names or licensed font files.
- Brand voice: short words that describe the product tone.

## Lobby And Room Cards

- Default room cover images for music, video, group video, solo live, PK live, private, and password rooms.
- Optional themed backgrounds: neon, studio, midnight, mint.
- Empty-state illustration or image for no rooms.
- Demo host/avatar images for seeded users.

## Live RTC Room

- Default user avatar image or avatar style reference.
- Mic seat icons or preferred icon style if not using generated UI icons.
- Camera-off placeholder visual.
- Audio-only room visual.
- Connection-status visuals if you want custom branded states.
- Gift/effects/screen-share icons or product-specific references.

## Chat

- Empty chat visual.
- Message state icons: sent, delivered, unsent/deleted, moderation notice.
- Optional reaction/gift images.

## Admin Dashboard

- Product logo mark for admin sidebar.
- Metric/monitor icon style references.
- Empty-state image for no active sessions.
- Brand-specific status colors for verified, live, warning, failed.

## SDK Page

- SDK/product wordmark.
- Developer diagram style reference.
- Code sample screenshot style, if there is a preferred docs brand.

## Marketing/Store Readiness

- App screenshots: desktop lobby, live room, admin dashboard, SDK page.
- Short product tagline.
- One-paragraph product description.
- Target customer/persona.
- Competitor/style references: 2-5 links or screenshots.

## File Requirements

- SVG for logos/icons where possible.
- PNG/WebP for photos or illustrations.
- 2x resolution for raster assets.
- Transparent background for avatars, logos, and icons when possible.
- Confirm licensing/permission for every asset.
